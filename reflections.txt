# What you added to Udacitask 2.0
# -------------------------------------------------------
1. What is the gem of your choice and how did you use it?
I added the gem 'terminal-table' and used it to display
a few statistics about the todo lists that have been created.

2. Describe the 2 features you added to the project.
  a) It is now possible to change the priority of a todo item
  b) It is now possible to set todo items to completed. They are
     shown with a little checkmark in the list output.
